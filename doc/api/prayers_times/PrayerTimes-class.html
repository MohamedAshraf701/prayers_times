PrayerTimes

PrayerTimes class

The PrayerTimes class calculates prayer times based on provided coordinates, date, and calculation parameters.

This class uses the CelestialMath and StellarMoment classes for calculations.

The calculations are based on established CelestialMath algorithms and formulas. Please refer to authoritative sources for detailed explanations and references for the calculations.
